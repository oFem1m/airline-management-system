# Фронтенд для системы управления авиакомпанией

Команды выполнять в директории frontend

Для корректной работы требуется npm

Также необходимо создать файл .env (см. [env.md](.env))

## Запуск

---

### Установка необходимых пакетов

```bash
npm install
```

---

### Запуск

```bash
npm run dev
```
